# 📊 Financial Analytics Dashboard

## 📌 Project Overview

This project presents an end-to-end Financial Analytics Dashboard built using Python, SQL, and Power BI. The objective is to analyze supermarket sales transactions and generate meaningful financial insights through interactive visualizations and business intelligence techniques.

---

## 🎯 Project Objectives

* Clean and preprocess raw financial data
* Analyze revenue, expenses, and profit
* Identify business trends and patterns
* Create an interactive Power BI dashboard
* Generate actionable business insights

---

## 📈 Key Performance Indicators (KPIs)

* 💰 Total Revenue
* 💸 Total Expense
* 📈 Total Profit
* 📊 Profit Margin
* 🛒 Revenue by Product Line
* 🏙️ Profit by City
* 💳 Revenue by Payment Method
* 📅 Monthly Revenue Trend
